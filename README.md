# maxscale-helmChart

maxscale helm chart for mariadb replication 



| Parameter                 | Description                                     | Default                                                 |
|---------------------------|--------------------------------------------------------|--------------------------------------------------|
| ` server1 `               | server1 address                                        | `master`                                         |
| ` server2 `               | server2 address                                        | `slave1`                                         |
| ` server3 `               | server3 address                                        | `slave2`
